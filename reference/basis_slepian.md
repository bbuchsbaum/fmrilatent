# Basis specifications (lightweight descriptors)

Basis specifications (lightweight descriptors)

## Usage

``` r
basis_slepian(k = 3, type = "laplacian")

basis_pca(k = 3, whiten = FALSE)

basis_flat()
```

## Arguments

- k:

  Number of components.

- type:

  Basis flavor (implementation-dependent).

- whiten:

  Whether to whiten PCA scores.
