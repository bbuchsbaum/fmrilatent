# fmrilatent: Latent Space Representations of fMRI Data

Provides S4 classes and methods for representing neuroimaging data in
latent spaces. The LatentNeuroVec class offers a memory-efficient
representation using matrix factorization (basis x loadings), enabling
compact storage and efficient computation for dimensionality-reduced
fMRI data such as PCA or ICA decompositions. Integrates with the
neuroim2 package for standard neuroimaging data structures.
