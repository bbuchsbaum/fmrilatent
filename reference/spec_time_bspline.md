# Temporal B-spline spec

Temporal B-spline spec

## Usage

``` r
spec_time_bspline(
  k,
  degree = 3L,
  include_intercept = FALSE,
  orthonormalize = TRUE
)
```

## Arguments

- k:

  Components (df).

- degree:

  Spline degree (default 3).

- include_intercept:

  Logical include intercept.

- orthonormalize:

  Logical orthonormalize columns (default TRUE).

## Value

A \`spec_time_bspline\` object.
