# Spatial wavelet (active pencil) spec

Spatial wavelet (active pencil) spec

## Usage

``` r
spec_space_wavelet_active(levels_space = 2L, levels_time = 0L, threshold = 0)
```

## Arguments

- levels_space:

  Spatial lifting levels.

- levels_time:

  Optional time lifting levels.

- threshold:

  Threshold after space transform.

## Value

A \`spec_space_wavelet_active\` object.
