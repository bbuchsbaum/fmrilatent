# Spatiotemporal spec (separable)

Spatiotemporal spec (separable)

## Usage

``` r
spec_st(time, space, core_mode = c("auto", "explicit"))
```

## Arguments

- time:

  Temporal spec (\`spec_time\_\*\`).

- space:

  Spatial spec (\`spec_space\_\*\`).

- core_mode:

  Reserved (currently "auto").

## Value

A \`spec_st\` object.
